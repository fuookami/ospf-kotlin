# 货物（Cargo）

:us: [English](README.md) | :cn: 简体中文

负责航班恢复调度系统中货物域操作的限界上下文。此上下文目前是未来货物特定逻辑（如货物路由、容量跟踪和中断处理）的占位符。

## 职责

- 管理货物特定的领域对象和操作。
- 跟踪航班任务的货物容量。
- 处理航班恢复期间的货物中断场景。

## 核心类

| 类名 | 说明 |
|---|---|
| `CargoContext` | 货物域操作的入口上下文。 |
| `Aggregation` | 货物域对象聚合（占位符）。 |

## 依赖

- **task** — 提供 `FlightTask`、`Aircraft`、`AircraftCapacity.Cargo`。
- **rule** — 提供货物相关的流量控制和限制。
